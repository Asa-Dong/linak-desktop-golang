# Linak Desk Control for GNOME

[中文版 (Chinese Version)](README_zh.md)

A GNOME Shell extension and Go daemon for controlling Linak Bluetooth standing desks. Generated by Antigravity.

![Desktop Screenshot](screenshot.png)

## Features

- **Native Integration**: Integrated into the GNOME Quick Settings menu with a clean, responsive UI.
- **Precise Control**: Supports continuous movement on button hold and immediate stop on release.
- **Session Managed**: Automatically starts and stops with the GNOME session via Systemd.
- **Real-time Stats**: Displays real-time desk height when the Quick Settings menu is opened.

## Installation

Run the provided installation script to compile the Go daemon, register the Systemd service, and install the GNOME extension:

```bash
chmod +x install.sh
./install.sh
```

**Post-installation:**

1. **Logout and login** (or restart) to load the new GNOME Shell extension.
2. Enable the extension via terminal or the "Extensions" app:
   ```bash
   gnome-extensions enable linak-desk@asa.github.io
   ```
3. Use the "Desk Control" section in the Quick Settings menu (top-right corner).

## Architecture

- **Go Backend**: Built using `tinygo.org/x/bluetooth` for low-latency BLE communication and `godbus` for IPC. Includes memory-safe concurrency and automatic connection retries.
- **GNOME Extension**: A GJS-based extension that interacts with the daemon over DBus, using native `button-press-event` and `button-release-event` signals for precise tactile control.
